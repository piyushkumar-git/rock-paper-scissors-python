# Rock Paper Scissors Game (Python)

This is a simple Rock Paper Scissors game I built while learning Python functions.

I wrote this project myself as practice to improve my understanding of:

* Functions
* Loops
* Conditionals
* User input handling
* Basic game logic

## How it works

The user selects:

* Rock (r)
* Paper (p)
* Scissor (s)

The computer randomly selects one option.

The program compares both choices and shows:

* Win
* Lose
* Tie

The user can continue playing multiple rounds.

## Features

* Input validation
* Random computer choice
* Clean function-based structure
* Emoji display for better experience
* Infinite play until user exits

## Technologies used

* Python 3
* random module

## How to run

1. Install Python 3
2. Download the file
3. Run:

```
python rps.py
```
